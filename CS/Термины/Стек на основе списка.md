**Стек на основе списка** - это реализация структуры данных [[Стек]], где каждый элемент (узел), содержит данные и указатель на следующий элемент ([[Список]], [[Связный список]])

В данном случае, размер стека не предсказуем.

Структура:
```c
typedef struct Node {
	int data;
	struct Node* next;
} Node;

typedef struct Stack {
	Node* top;
} Stack;
```

### Алгоритмы
#### Push
Суть: новый элемент становится новым головным узлом, а его ссылка указывает на бывшую голову.
Сложность: O(1)
```c
void push(Stack* stack, int value){
	Node* new_node = malloc(sizeof(Node));
	new_node->data = value;
	new_node->next = stack->top;
	stack->top = new_node;
}
```

#### Pop
Суть: удаляется текущий головной узел, а ссылка на голову переносится на следующий узел.
Сложность: O(1)
```c
int pop(Stack* stack, int value){
	Node* temp = stack->top;
	int popped_data = temp->data;
	stack->top = temp->next;
	free(temp);
	return popped_data;
}
```

#### Peek/Top
Суть: возвращение значение головного узла.
Сложность: O(1)
```c
int peek(Stack* stack){
	if(!stack) return -1;
	return stack->data;
}
```
