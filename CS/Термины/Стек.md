**Стек** - [[Абстрактная структура данных]], представляющая собой список элементов, организованных по принципу **LIFO**
(last in - first out)
Стек можно представить в виде стопки любый предметов, например книг:
![[Pasted image 20260430093812.png|218]]

На картинке ниже схематично изображен стек. Блок вида "Данные/\*next" - ячейка данных в стеке.
\*next - указывает на следующим элемент, то есть хранит адрес следующей ячейки.
Указатель \*top указывает на вершину стека, то есть хранит её адрес.
