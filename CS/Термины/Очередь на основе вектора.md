**Очередь на основе вектора** - это структура данных [[Очередь]], работающая по принципу FIFO и на основе [[Вектор]].

Структура:
```c
typedef struct Queue {
	int* data;
	int head;
	int tail;
	int size; // текущее количество элементов
	int capacity;
} Queue;
```

### Алгоритмы
#### Push Back (вставка в очередь)
Суть: Если массив заполнен (`size == capacity`), создаётся новый массив вдвое большего размера, все элементы копируются в него по порядку начиная с `head`, а `head` и `tail` перенастраиваются на начало. Затем `tail` сдвигается циклически на одну позицию, туда записывается новое значение, `size` увеличивается. 
Сложность: амортизированное время O(1). В худшем случае: O(n)
```c
void push_back(Queue* q, int value) {
    if (q->size == q->capacity) {
        int newCapacity = q->capacity * 2;
        int* newData = (int*) malloc(newCapacity * sizeof(int));
        for (int i = 0; i < q->size; i++) {
            newData[i] = q->data[(q->head + i) % q->capacity];
        }
        free(q->data);
        q->data = newData;
        q->head = 0;
        q->tail = q->size - 1;
        q->capacity = newCapacity;
    }
    q->tail = (q->tail + 1) % q->capacity;
    q->data[q->tail] = value;
    q->size++;
}
```

#### Erase (удаление)
Суть: запоминает значение по индексу `head`, затем циклически сдвигает `head` вперёд, уменьшает `size` и возвращает сохранённое значение.
Сложность: O(1)
```c
int erase(Queue* q) {
    if (isEmpty(q)) {
        return -1;
    }
    int value = q->data[q->head];
    q->head = (q->head + 1) % q->capacity;
    q->size--;
    return value;
}
```

#### Front (возвращает первый элемент очереди без удаления)
Суть: при непустой очереди возвращает элемент вектора по индексу `head`
Сложность: O(1)
```c
int front(Queue* q) {
    if (isEmpty(q)) {
        return -1;
    }
    return q->data[q->head];
}
```
