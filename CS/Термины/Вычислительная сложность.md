**Вычислительная сложность** - понятие в CS, обозначающее функцию зависимости объема работы, которая выполняется некоторым алгоритмом, от размера входных данных.

**Объём  работы** обычно измеряется абстрактными понятиями времени и пространства, называемыми вычислительными ресурсами.

**Время** опредялется количеством элементарных шагов, необходимых для решения задачи.

Основные типы сложности:
1. Временная сложность (Time Complexity): сколько времени (или операций) потребуется для работы алгоритма.
2. Ёмкостная сложность (Space Complexity): сколько дополнительной памяти потребуется алгоритму.

Для оценки сложности алгоритмов используется нотация [[Big O]], которая показывает верхнюю границу времени выполнения или использования памяти, выраженную как функцию размера входных данных n.
