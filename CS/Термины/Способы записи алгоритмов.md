1. Словесный (вербальный).
   Пошаговое описание алгоритма на обычном языке.
   Пример: 1) присвоить значение 2) умножить на 2 3) вывести результат
2. Графический (Блок-схемы).
   Изображение алгоритма с использованием геометрических фигур.
   Пример: 
   [[Pasted image 20260427204716.png]]
3. Псевдокод. Полуформализованное описание, сочетающее элементы ествественнного языка и языков программирования.
   Пример:
   ```python
   if a > b THEN
	   max = a 
   ```
4. Программный код.
   Запись алгоритма на языках программирования для исполнения компьютеров.
5. Табличный. Используется для алгоритмов с четкой зависимостью между входными и выходными данными, оформляется в виде таблицы.
