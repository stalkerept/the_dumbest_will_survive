
**Отображение структуры данных в памяти списком** - это списковое представление структуры данных, то есть элементы разбросаны по памяти и соединяются указателями. Такой набор узлов и называют [[Список]].

![[Pasted image 20260501200135.png]]

Пример узла на основе односвязного списка:
```c
typedef struct Node {
	int data;
	struct Node* next;
} Node;
```
